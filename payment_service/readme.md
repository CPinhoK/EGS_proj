# Running Info

## Payment Frontend

### Prerequisites
NodeJS https://nodejs.org/en/

- Install node modules
```bash 
npm install --force
```
- Run React app
```bash 
npm start
```
## Payment Api

### Prerequisites
Python 3.10 https://www.python.org/downloads/

- Install python libraries
```bash 
pip install -r requirements.txt
```
- Run Fastapi api
```bash 
uvicorn main:app 
```
